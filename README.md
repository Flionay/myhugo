## 记录Hugo博客2.0

Hugo的优点就是基于Go语言开发，生成静态文件速度很快。随着用户量的增加，主题也逐渐多样，也就造成了参差不齐的主题水平，一个集合所有功能，而且美观的主题一直没有找到。图片主题是基于Allinone的，自己也进行了很多优化，添加了一些功能.

如果想要用Hugo的朋友可以去借鉴我的Hugo标签文章，记录了优化过程，最新[博客](https://www.angyi.online/)。
